HostFact [version] backup

Er is een fout opgetreden tijdens de laatste poging om een backup van de database te maken. Controleer uw server error log om de oorzaak te achterhalen, of schakel de backup functionaliteit van HostFact zelf uit.

Mogelijke oorzaken:
• het memory_limit is bereikt.
• een timeout is geraakt, het maken van de backup duurt te lang.


Dit is een automatisch verzonden e-mail vanuit uw HostFact
